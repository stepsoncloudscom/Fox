## 1. Nerede kaldık (30 sn)
Yeni müşteri; **sıfırdan marka inşası**. Döneminin öncüsü, 20 yıl usta, ama çağın gerisinde + idari cezalar → temiz sayfa. Hedef: **daha çok hasta değil → itibar + yurtdışı hasta** (döviz). Ticari yapı netleşti: tek seferlik (Kurumsal Çekirdek + Sistem kuruluş) + aylık (strateji + görsel içerik). Tüm belgeler hazır.

## 2. Masaya koyacakların (elinde hazır)
- **Çerçeve:** Kurumsal Çekirdek + Sistem (görselle anlat — sistem mantığı)
- **12 aylık plan + timeline** (çekim öne alındı; CP0–CP6 checkpoint'ler)
- **Referans yaklaşımı:** Össur/Ottobock onur-merkezli temsil — *müşteriye "sizi onlar gibi yapacağız" değil,* "onurlu + premium + uluslararası" dili
- **İş kalemleri + bütçe formu** (ajans kalemlerine göre; bedeller senin)

## 3. SEN kapatmalısın (görüşme öncesi/sırasında)
- **Para birimi:** TL mi döviz mi? (yurtdışı hasta → döviz lehine, §11.1)
- **Aylık içerik adedi** → aylık bedeli belirler
- **Bedeller** (çekirdek / web / sosyal / çekim / aylık) → bütçe formuna
- **İsim korunur** onayı (öneri: evet — kurucu/usta sermayesi)
- **Tek master-brand** onayı (ozgurprotez + biyonikprotezler birleşir)

## 4. MÜŞTERİDEN al (context boşlukları — `[DOĞRULANACAK]`)
- **Hedef kitle gerçekliği + müşteri dili** (kendi kelimeleriyle — verbatim değerli)
- **İş hedefleri** somutlaştır (yurtdışı hangi ülke/segment; hacim beklentisi)
- **Mevcut görsel varlıklar / marka tercihleri** (neyi seviyor, neyi istemiyor)
- **⚖️ Hukuki durum:** cezalar/uyum **geçmişte mi kapandı, süren mi?** Avukatı kim? → hukuk gate kurulumu buna bağlı
- **Web neden bakımda** + iki domain durumu (birleşme için)
- **Hasta hikâyeleri + izin:** çekim erken başlıyor → hangi hasta(lar), izin süreci

## 5. Nöbet / hassasiyet (aklında tut)
- **🕊 Onur gerilimi:** müşteri "kolay/düzgün kullanıcı" diyebilir — bu onun ticari tercihi; ama **premium'u hizmet+teknoloji üzerinden kur, seçicilik dışarı sızmasın** (§11.8). Dışarıya "zor vakayı elemeyiz" imajı = itibar riski.
- **⚖️ Hukuk gate:** web + her medikal iddia avukat onayından geçer; Sağlık Bakanlığı reklam mevzuatı sert → "hemen agresif reklam" beklentisini yönet.
- **💱 Telif:** yurtdışı = global kullanım telifi (film/foto margin kaldıracı — düşük vermeyelim).
- **⏳ Beklenti:** SEO/itibar 6–9 ayda olgunlaşır; "hemen yurtdışı hasta" değil, sistem kurulumu.

## 6. Kapanış hedefi / sonraki adım
Teklif kabulü → sözleşme + **avukatla hukuk çerçevesi** → **Hafta 0 başlar** (çekim hazırlığı + hasta izni en yavaş parça, hemen açılır).
