## B · ÜRETİM ADIMLARI (6 faz)

### Faz 0 — Ön Koşullar (kapı: proje aktivasyonu)
- [ ] **Sözleşme imzası + D1 %50 avans** (kimlik işi bununla resmen başlar — Kademe 3, Ayhan)
- [ ] Avukat düzeltmeleri işlendi mi → IP maddesi kesin hali = §12'nin kaynağı
- [ ] Tagline B kıdem teyidi ("1996'dan beri" belge kanıtı — müşteriden)
- [ ] Baskı/ekran renk kalibrasyon testi planı (hex sabitleme)

### Faz 1 — Logo Craft (kritik yol — en uzun teslim)
- [ ] Tasarım brief'i dış üreticiye: kilitli kararlar belgesi + yasak formlar listesi ([[logo-vektor-uretimi-fox-yapmaz]] — Fox konsept/brief/denetim yapar, final craft insanda)
- [ ] 2-3 yön → Fox ön eleme (kilitli kurallara uygunluk) → Ayhan seçimi → müşteri onayı
- [ ] **TÜRKPATENT + uluslararası benzerlik taraması** (kanatlı-ayak kalabalık alan — tescil öncesi zorunlu, avukat)
- [ ] Varyasyon seti: dikey/yatay/ikon + AR/RU lockup + mono pozitif/negatif

### Faz 2 — İçerik Yazımı (logo ile paralel yürüyebilir)
- [ ] Bölüm 2-3-10-11 metinleri: kilitli belgeden brand book diline (Branding Ajanı)
- [ ] Bölüm 4 Copywriting: kelime listeleri → kanal-bazlı kalıp + örnek metinler (İçerik Ajanı girdisiyle)
- [ ] Bölüm 12 Legal: sözleşme kesin IP maddesinden birebir (çelişki = Denetmen DUR)

### Faz 3 — Görsel Üretim & Mockup
- [ ] Moodboard/referans kare derlemesi (görsel dil ilkelerine örnek)
- [ ] Uygulama mockup'ları (Bölüm 9) — logo finali geldikten sonra
- [ ] Yanlış kullanım örnekleri görselleştirme (Luxmed dersi: bu bölüm atlanmıştı, bu kez planlı)

### Faz 4 — Belge Üretimi
- [ ] `pdf-motoru.py`'ye Özgür Irmak teması (Eksen 15 palet + Plex/Lora) eklenir — SOC imzası geride
- [ ] Tam render → render-and-review checklist (görsel standart §7) + 4-dil glyph/RTL testi
- [ ] Kara liste taraması: placeholder sızıntısı + söylem-görsel çelişkisi ("belgenin verdiği kuralı belgenin kendisi geçiyor mu?")

### Faz 5 — Denetim & Teslim Zinciri
- [ ] **Denetmen (Faz 3 filo):** Verification (rakam/tarih/lisans/madde — özellikle §12 sözleşme senkronu) + Red Team (üçüncü-taraf kullanım koşulları stres testi) + orkestratör sentezi (tutarlılık/kalite/onur)
- [ ] **Avukat gate:** dışa dönük dil öğeleri (tagline dahil) + §11-12
- [ ] **Ayhan onayı** → müşteri sunumu (Kademe 2)
- [ ] Kabul sonrası: kaynak dosyalar + kullanım devri (sözleşme finaline bağlı)

## C · KRİTİK YOL & AÇIK KALEMLER

**Kritik yol:** Sözleşme imzası → logo craft (dış üretici + tescil taraması) → mockup'lar → tam belge. Logo en uzun kalem; Faz 2 (metin) beklemeden paralel başlar.

**Ayhan'da:** ① imza/aktivasyon ② logo dış üretici seçimi (öneri istenirse Fox liste çıkarır) ③ tagline B kıdem belgesi müşteriden isteme.
**Avukatta:** IP maddesi finali · tescil taraması · dil öğeleri gate.
**Fox/filoda:** brief paketi (Faz 1) + metin bölümleri (Faz 2) — imza sonrası ilk 48 saatte hazır olacak şekilde.
