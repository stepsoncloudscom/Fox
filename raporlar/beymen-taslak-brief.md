# Beymen Taslak — Brief
*Yöntem: Ayhan doneleri parça parça verir → "bitti" → varyasyon üretilir → tek tek onaylanır. Üst-segment outreach, analiz-temelli, zamana yayılır.*
*REGISTER (kilitli): Beymen üst segment — ölçülü, rafine, ağırlığı olan dil. Lakayıt / fazla-samimi / "peşinen söyleyeyim" tonu YASAK.*
*MARKA MİMARİSİ (sabit — bir daha sorma): **Steps on Clouds = yaratıcı yönetim ajansı** (üst marka). **Marka Bulutu = ajansın sistemi/yaklaşımı.** **Ayhan Erden = kurucu / yaratıcı yön.** **Fox = yapay zekâ ajanımız** (hız+ölçek; ruh/karar insanda). İmza: Ayhan Erden, Kurucu — Steps on Clouds.*

## Anlatı omurgası
Ayhan Erden + global deneyim → Steps on Clouds (yaratıcı yönetim ajansı) + insan-odaklılık → Marka Bulutu sistemi → çekirdek değer: görsel dilin pazarlamaya + hikâyeye dönüşümü. İnsan+AI: ruh/karar insanda, hız/ölçek Fox'ta.

## Onaylı yön: V1+V3 harman
Açılış MARKA-ÖNCE (önce marka, sonra biz). AI-çağı + Fox hafif dokunuş (V3) + ortak yaratım teklifi, markanın kendi hassasiyeti ekseninde (V1). İnsan önceliği korunur.

## Tema vurguları (gövdede, markaya göre dozla)
- Yapay zekânın hızı × insanın ruhu (ruh insanda)
- **Adaptif moda = kapsayıcı/erişilebilir "adaptive fashion"** (SoC insan-odaklı DNA). ⚠️ ONUR MERKEZLİ (§9): acıma/ilham pornosu yok. Kategoriye göre: giyim/mayoda doğrudan; takı/objet/çantada "insan ruhu / dokunduğu yer" üzerinden.
- Hikâye anlatımının önemi
- Herkesleşmeye karşı markanın kendi sesi + "dokunduğu yer"

## Özelleştirme & tempo
- Her mail markaya özel ANALİZ'den doğar (görsel dünya, konum, kendi hassasiyeti, açıklar) — yüzeysel değil.
- Zamana yayılır: topluca gönderim yok; marka-marka, ölçülü tempo.
- Akış: marka seç → analiz → şablonu uyarla → Ayhan onayı → sıradaki.
